# Navo 5 Stable Rebuild

هذه نسخة إعادة بناء كاملة ومختلفة عن النسخ السابقة، الهدف منها حل مشاكل:
- القائمة
- الداشبورد
- المهام
- الوقت
- التركيز
- المساحات
- الإعدادات
- تسجيل الدخول
- ربط بيانات كل مستخدم

## ماذا تغير؟

### 1) نظام تسجيل دخول واضح
- إنشاء حساب باسم مستخدم وكلمة مرور.
- تسجيل دخول.
- تجربة سريعة Demo.
- كل مستخدم له بياناته الخاصة في LocalStorage.

### 2) نظام بيانات أفضل
- البيانات محفوظة بهذا الشكل:
  - قائمة المستخدمين.
  - بيانات منفصلة لكل مستخدم.
  - جلسة دخول محفوظة عند تفعيل “تذكرني”.

### 3) واجهة جديدة بالكامل
- قائمة جانبية مرتبة.
- قائمة جوال سفلية.
- داشبورد جديد.
- بطاقة وقت مباشرة.
- AI Pulse محلي يعطي اقتراحات حسب المهام.

### 4) المهام
- إضافة / تعديل / حذف / إنجاز.
- بحث.
- فلترة.
- ترتيب ذكي.
- ربط المهمة بالمساحة والأولوية والتاريخ.

### 5) التركيز
- مؤقت تركيز.
- تشغيل / إيقاف / إعادة.
- Deep Focus Mode.
- اختصارات:
  - Space تشغيل/إيقاف
  - Enter تشغيل
  - Esc وضع عميق

### 6) المساحات
- إضافة مساحة.
- حذف مساحة.
- لون لكل مساحة.
- ربط المهام بالمساحات.

### 7) البروفايل
- اسم عرض.
- نبذة.
- XP.
- Level.
- إحصائيات المستخدم.

### 8) الإعدادات
- تبديل الوضع الليلي والنهاري.
- تغيير لون الهوية.
- وضع أداء خفيف.
- تصدير واستيراد البيانات.
- تصفير بيانات المستخدم.

## التشغيل
افتح المشروع في VS Code ثم شغل Live Server على index.html.

## ملاحظة مهمة
هذه نسخة Local-first ثابتة. Supabase موجود داخل مجلد supabase/config كمرحلة لاحقة، لكن الحفظ الأساسي الآن محلي وآمن لكل مستخدم حتى ما تتلخبط البيانات.
