# Navo v6 Luxury Workspace

نسخة جاهزة للتشغيل من Navo بترتيب ملفات صحيح وهوية أفخم.

## التشغيل
افتح `index.html` عبر Live Server في VS Code.

## المسارات بعد التنظيم
- `index.html`
- `styles/main.css`
- `app/main.js`
- `config/navo-config.js`
- `assets/navo-mark.svg`
- `assets/navo-icon.png`
- `manifest.webmanifest`
- `sw.js`

## وش تم ضبطه
- إعادة ترتيب المشروع بمجلدات واضحة.
- إصلاح مسارات CSS / JS / Assets / Manifest.
- تحسين هوية Navo v6 بطبقة Luxury Motion.
- تحسين الواجهة الزجاجية، الظلال، العمق، الخلفية، والـ hover.
- تحسين شاشة الدخول والموبايل.
- دعم تسجيل دخول محلي باسم مستخدم وكلمة مرور.
- حفظ بيانات كل مستخدم بشكل منفصل في LocalStorage.
- داشبورد، مهام، تركيز، مساحات، بروفايل، وإعدادات.
- تصدير واستيراد البيانات.

## ملاحظة
Supabase موجود كملف تجهيز لاحق `SUPABASE_SETUP.sql`، لكن النسخة الحالية Local-first وجاهزة بدون إعدادات خارجية.
